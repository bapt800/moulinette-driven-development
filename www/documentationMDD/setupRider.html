Set Up de Rider.

  • Cloner le repos GitHub
  • Ouvrer la solution de Test voulu avec Rider. Ici Test/Test_StarterPack
    Avec Test/Test_StarterPack/TestCalculator comme solution de Test
    Et Test/Test_StarterPack/Calculator , comme votre projet perso

    On ouvre la solution de test. Pour l'instant Rider indique qu'il y a des références manquantes, l'on peut déja naviger dans les tests.
    Pour links son projet aux Tests, faire un clic droit sur la solution de Test (avec le logo "C#"), puis "add" -> "add existing items..." -> choisir le bon ficher.cs de son proj perso -> "add links"
    Faire cela pour touts les fichiers demandés, enfin l'on peut executer les tests.

Lien vers le GitHub : Ce site est automatiquement sync sur la branche master du github.
Cela demande moins de travail de maintenance...
